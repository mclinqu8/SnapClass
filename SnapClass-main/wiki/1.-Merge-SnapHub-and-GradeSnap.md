## 1.1 Integration
The aim is to merge existing Snap! support systems- SnapHub and GradeSnap into one environment for total classroom support.

## 1.2 SnapHub
* Teachers and students should be able to create accounts in Cellular and iSnap
* The environment should include intelligent features such as hints, feedback, and examples that are already incorporated into the system

## 1.3 GradeSnap
* GradeSnap should provide a rubric grading environment
